VISUAL STUDIO CODE

Visual Studio Code (VS Code) is a lightweight but powerful source code editor developed by Microsoft. It supports a wide range of programming languages and comes with features.

Features of VScode:

  • Syntax Highlighting and Auto-Completionfor most languages
  • Integrated Terminalfor running scripts or commands.
  • Version Control Integrationfor Git and other systems.
  • Extensions Marketplaceto add functionality, such as language-specific support, linters, debuggers, themes, and more.
  • Customizable interfacewith user-defined settings and shortcuts.
  • Built-in-Debuggingfor a seamless coding experience.

    Let us know about some shortcuts.

    Emmet symbol Description
    Ctrl+s Save the current file.
    Ctrl+o Opens the selected file.
    Ctrl+c Copy the selected text
    Ctrl+v Paste the selected text.
    Ctrl+x Cut.
    Ctrl+z Cut the selected text or item.
    Ctrl+y Redo the previously undone action.
    Ctrl+a select all content in the current file.
    Ctrl+Shift+p Open command palette.
    Ctrl+Shift+n New window.
    Ctrl+Shift+w Close window
    Ctrl+Shift+Alt+Down Add cursors to multiple lines for simultaneous editing.
    Alt+UpArrow Move the selected line or block of code one line up.
    Alt+DownArrow Move the selected line or block of code one line down.
    Ctrl+p Prints a document.
    Shift+Delete Deletes a selected item permanently.
    F2 Renames a selected item.
    F5 Refreshers the current window or webpage.
